# 引用类型和动态类型
#引用类型

##### object

		引用类型不包含存储在变量中的实际数据，但他们包含对变量的引用。
		可以理解为引用类型定义一块空间，会按照存储在这块空间中的数值去改变自身的数据类型。对引用类型的数值进行赋值，这个过程称为装箱。

##### dynamic

		动态类型差不多一个意思，但引用类型赋值后就会定型，而动态不会

##### String

		非常方便的字符串类型在C sharp中也有。
